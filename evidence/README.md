# evidence/

Reproducible proof of the attacks. Generated by [`../evidence.sh`](../evidence.sh).

- **`solved-challenges.md`** — *committed.* Which Juice Shop challenges are solved,
  pulled from `/api/Challenges` (the gamified confirmation, as a diff-able table).
- **`files/`** — *git-ignored.* Raw downloads + responses (the SQLi token, the
  `/ftp` `.bak` files, `challenges.json`). Kept local as proof, not committed
  (binaries / planted secrets).
- **`attack-<ts>.log`** — *git-ignored.* Full `attack.sh` output for the run.

Regenerate from Kali:

```bash
TARGET=http://<mac-ip>:8088 ./evidence.sh
```

Screenshots (XSS alert firing, Score Board UI) go in `../writeups/assets/` — see
[../EVIDENCE.md](../EVIDENCE.md).
